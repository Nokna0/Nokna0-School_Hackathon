# EduTech 로컬 실행 가이드

## 📋 필수 요구사항

- Node.js 18 이상
- npm 또는 pnpm
- MySQL 데이터베이스 (또는 제공된 .env.local의 TiDB 사용)

---

## 🚀 빠른 시작 (3단계)

### 1단계: 의존성 설치
```bash
npm install
```

### 2단계: 데이터베이스 초기화
```bash
npm run db:push
```

### 3단계: 개발 서버 실행
```bash
npm run dev
```

브라우저가 자동으로 열리지 않으면 `http://localhost:5173` 접속

---

## ✅ 확인 사항

### 정상 작동 시:
- ✅ 홈페이지에서 영어, 수학, 화학 과목 선택 표시
- ✅ 로그인 없이 바로 과목 클릭 가능
- ✅ 수학 페이지에 PDF 파일 3개 표시
- ✅ PDF 업로드 버튼 작동

### 문제 발생 시:
1. **"Cannot find module" 에러**
   ```bash
   rm -rf node_modules package-lock.json
   npm install
   ```

2. **데이터베이스 연결 에러**
   - `.env.local` 파일이 있는지 확인
   - `DATABASE_URL`이 올바른지 확인

3. **포트 충돌 (EADDRINUSE)**
   ```bash
   # Windows
   netstat -ano | findstr :5173
   taskkill /PID <PID번호> /F
   
   # Mac/Linux
   lsof -ti:5173 | xargs kill -9
   ```

## 🔧 Groq API 설정 (선택사항)

AI 기능(퀴즈 생성, 분석)을 사용하려면:

1. https://console.groq.com/ 접속
2. 무료 API 키 발급
3. `.env.local` 파일 수정:
   ```
   GROQ_API_KEY=your_api_key_here
   ```

## 📁 프로젝트 구조

```
EduTech/
├── client/          # 프론트엔드 (React + Vite)
│   ├── src/
│   │   ├── pages/   # 페이지 컴포넌트
│   │   └── components/
├── server/          # 백엔드 (Express + tRPC)
├── drizzle/         # 데이터베이스 스키마
└── .env.local       # 환경 변수
```

## 🎯 주요 기능

- **PDF 업로드 및 보기**: 학습 자료를 업로드하고 페이지별로 읽기
- **AI 분석**: 수학 공식, 영어 단어 자동 분석
- **퀴즈 생성**: PDF 내용 기반 자동 퀴즈 생성
- **학습 기록**: 공부한 내용 저장 및 복습

---

## 💡 팁

- **개발 서버 재시작**: `Ctrl + C` 후 `npm run dev`
- **데이터베이스 리셋**: `npm run db:push` 재실행
- **빌드**: `npm run build` (프로덕션용)
